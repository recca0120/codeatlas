# CodeAtlas

[![Build and Deploy](https://github.com/recca0120/codeatlas/actions/workflows/deploy.yml/badge.svg)](https://github.com/recca0120/codeatlas/actions/workflows/deploy.yml)
[![Collect Data](https://github.com/recca0120/codeatlas/actions/workflows/collect.yml/badge.svg)](https://github.com/recca0120/codeatlas/actions/workflows/collect.yml)

全球開發者排行榜 — 追蹤 **138 個國家**最活躍的 GitHub 使用者，依公開貢獻、總貢獻及追蹤者排名。每週更新。

**線上網站：** https://recca0120.github.io/codeatlas/

## 功能

- 國家級開發者排名（公開貢獻、總貢獻、追蹤者）
- 互動式 3D 地球儀視覺化
- 個人開發者頁面（程式語言分佈、熱門儲存庫）
- 依城市、程式語言篩選，支援使用者名稱搜尋
- 國際化（英文 / 繁體中文）
- 深色 / 淺色模式
- Google Analytics 整合
- 透過 GitHub Actions 每週自動收集資料
